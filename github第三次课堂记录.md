# 第三次课堂记录
**1. MarkDown是什么**

Markdown 是一个 Web 上使用的文本到HTML的转换工具，可以通过简单、易读易写的文本格式生成结构化的HTML文档。

**2.  MarkDown的特点**
 * 纯文本内容，兼容所有的文本编辑与字处理软件
 * 可以放到版本管理系统中，追踪历史变更
 * 简介 高效 可读 直观 学习成本低
 * 专注你的文本内容而不是排版样式，安心写作
**3. MarkDown的用途**

写日志 技术文稿 记录代码片段 撰写文档  求职简历 等
